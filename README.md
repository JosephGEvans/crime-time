# crime-time
Exploring violent crime time of day occurrences for my WGU Data Analytics Capstone project 

## Setup

1. Install R from https://cran.r-project.org/
2. Install RStudio from https://posit.co/downloads/
3. Open RStudio
4. From the Console run `> install.packages("renv")`
5. Create a new project from the crime-time git repository

  > 1. File > New Project
  > 2. Version Control
  > 3. Git
  > 4. Repository URL: https://github.com/JosephGEvans/crime-time.git
  > 5. Create Project
  
6. In the new project, from the Console run `> renv::restore()`

  > Note: Downloading and installing all the dependency packages may take several minutes.

## Usage

 - Simply open the crime-time.R script and run it line-by-line.
 - Reproduce my results.
 - Play around with it to investigate other questions you may have.
